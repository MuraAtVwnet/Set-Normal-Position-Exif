Jpeg 画像方向を正位置にする

■ これは何?
 Jpeg 画像方向を正位置にします

■ 使い方
PowerShell プロンプトで以下コマンド入力すると、Jpeg の画像方向を正位置にします
オリジナルファイルは -ORG.jpg として残ります

SetNormalPositionExif ファイルPath

■ セットの仕方
PowerShell プロンプトで install.ps1 を実行してください

■ Uninstall 方法
uninstall.ps1 を実行して下さい

■ 動作確認環境
On Windows
	5.1
	6.2.1
	7.0.0-preview.5
